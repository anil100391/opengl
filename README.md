# Building
## Linux
* Open a terminal prompt in project directory
  ``` bash
      mkdir build
      cd build
      cmake -DCMAKE_BUILD_TYPE=Release ../.
      make
  ```
* Run the binary from project directory
  ``` bash
      ./build/app
  ```
